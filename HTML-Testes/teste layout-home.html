Banco de Dados ProxyMar

Seja bem-vinda(o) ao sistema de envio de dados do Laboratório de Proxies Marinhos para o Futuro do Oceano, do Instituto Oceanográfico da Universidade de São Paulo - o ProxyMar | IOUSP.

Este banco de dados foi criado com o objetivo de arquivar dados de projetos de pesquisa concluídos e em andamento do ProxyMar e de seus parceiros científicos, sem substituir a necessidade de compartilhamento dos dados de trabalhos científicos publicados pelo grupo de pesquisa em plataformas de dados abertos nacionais e internacionais. Por isso, para facilitar o compartilhamento futuro dos dados usamos os modelos para o envio de dados da plataforma Pangaea.de.

Antes de iniciar o processo de submissão, verifique aqui as diretrizes e toda a documentação necessária.
